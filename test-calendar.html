Тест календаря бронирования

Инструкции для тестирования:

  1. Откройте приложение по ссылке ниже
  2. Нажмите на кнопку "Забронировать столик" в Hero секции
  3. Найдите поле "Дата *" и нажмите на кнопку "Выберите дату"
  4. Проверьте, открывается ли календарь
  5. Попробуйте выбрать дату
Открыть приложение для тестирования

Возможные проблемы и решения:

  • Календарь не открывается: Проверьте консоль браузера на наличие JavaScript ошибок
  • Кнопка не реагирует: Возможно, проблема с CSS z-index или pointer-events
  • Календарь открывается, но не виден: Проблема с позиционированием или z-index

Проверка в консоли браузера:

Откройте DevTools (F12) и выполните в консоли:

// Проверка наличия элементов
